# BTI

##### What is this?
This program reads and displays data sent to a computer's serial port from our solar car's radio.

##### Dependencies
* PySerial
* PyQT (probably)

## Status

##### Completed Features:
* Connect to a serial device and read the output
* Parse the output from the radio
